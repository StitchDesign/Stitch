Highlights
- Vastly improved canvas navigation performance for graphs with many nodes.
- Optimized sidebar drag performance, especially for large numbers of items.

Improvements + What's New
- Introduced Spring Converter patch node.
- Introduced support for faster navigation between nodes with TAB and SHIFT+TAB.
- Introduced additional shortcuts for creating specific patch nodes.
- Added the ability to group and ungroup nodes with CMD+G and CMD+Delete shortcuts.
- Improved node menu accessibility by allowing menu to display over inspector view.
- Selecting node body now defocuses any focused inputs.
- Adjusted padding and margins of layer inspector on iPad.
- Improved project thumbnails when prototype uses dark background.

Bug Fixes
- Fixed issue where edges wouldn’t appear for new math expression inputs.
- Resolved issue where newly duplicated nodes were not staggered correctly on the canvas.
- Corrected issue where nodes in a group couldn’t be selected after being grouped.
- Fixed issues editing project titles.
- Improved shadow support for layer groups.
- Resolved issues related to MathExpression formula parsing numbers
- Layer Info patch node better supports loop of layers.
